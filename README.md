## 프로젝트 환경

### 1. 기술 스택

- 프레임워크 및 라이브러리
  - `Next.js`
  - `Framer Motion` : 애니메이션 라이브러리
- 스타일링
  - `Tailwind CSS` : 유틸리티 기반의 CSS 프레임워크
- 빌드 및 개발 도구
  - `npm` : 패키지 관리 도구, 의존성 설치 및 스크립트 실행을 관리

### 2. 개발 환경

```
- Node.js : v20.18.0
- npm
```
